Пройти интервью на решение задач просто

Все интервью стандартны. За один час надо дружески поговорить с интервьюером и решить две задачи по двадцать минут на каждую. Поэтому эти задачи - несложные.

Достаточно решить 200 задач из онлайн-задачника. Например, решайте задачи сложности 2 на Kattis.

Писать надо на Питоне - решения будут короче.

Дальнейшее чтение

Если вы прошли интервью по задачам, вы выиграли бесплатное путешествие в европейский офис компании. Так я посетил Цюрих, Лондон и Дублин. Можно из озорства пройти и он-сайт интервью, но это сложнее, чем научиться решать задачи.

Для интервью по устройству Linux надо прочитать Understanding the Linux Kernel. Можно найти перевод книги в Интернете бесплатно.

Для интервью по системному дизайну надо освоить Grokking the System Design Interview. Половина тем доступна бесплатно.

Что не надо делать

Не надо хотеть получить работу - так можно испортить себе настроение. Крутые компании мало нанимают. Намного проще найти работу по знакомству.

Не надо верить всему, что здесь написано. Это в чем-то субъективно, в чем-то устарело.
